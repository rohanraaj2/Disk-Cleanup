macOS DISK CLEANUP GUI - LAUNCH INSTRUCTIONS
============================================

To launch the Disk Cleanup GUI on macOS:

1. Double-click run.command in this folder
   OR
2. Open Terminal in this folder and run: ./run.command

Requirements:
- Python 3 installed
- tkinter library (usually included with Python)

The script will automatically navigate to the parent directory and run cleanup_gui.py

For questions or issues, see README.md in the main directory.

Note: You may also see Launch_Instructions.md which contains additional setup information.
