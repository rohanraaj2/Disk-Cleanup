macOS DISK CLEANUP GUI - LAUNCH INSTRUCTIONS
============================================

To launch the Disk Cleanup GUI on macOS:

1. Double-click DiskCleanupGUIApp.app in this folder (recommended)
   OR
2. Double-click run.command in this folder
   OR
3. Open Terminal in this folder and run: ./run.command

Requirements:
- Python 3 installed
- tkinter library (usually included with Python)

The launcher will automatically navigate to the parent directory and run cleanup_gui.py

For questions or issues, see README.md in the main directory.
