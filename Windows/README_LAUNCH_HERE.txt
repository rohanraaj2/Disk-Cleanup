WINDOWS DISK CLEANUP GUI - LAUNCH INSTRUCTIONS
==============================================

To launch the Disk Cleanup GUI on Windows:

1. Double-click run.bat in this folder
   OR
2. Open Command Prompt in this folder and run: run.bat

Requirements:
- Python 3 installed
- tkinter library (usually included with Python)

The script will automatically navigate to the parent directory and run cleanup_gui.py

For questions or issues, see README.md in the main directory.
