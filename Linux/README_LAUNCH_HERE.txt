LINUX DISK CLEANUP GUI - LAUNCH INSTRUCTIONS
============================================

To launch the Disk Cleanup GUI on Linux:

1. Double-click run.sh in this folder
   OR
2. Open terminal in this folder and run: ./run.sh
   OR
3. Copy UbuntuDiskCleanup.desktop to your desktop and double-click it

Requirements:
- Python 3 installed
- tkinter library (usually included with Python)

The script will automatically navigate to the parent directory and run cleanup_gui.py

For questions or issues, see README.md in the main directory.
